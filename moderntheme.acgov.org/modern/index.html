Read A Little Bit

Here you have your average, everyday text area. This one's pretty straight-forward, just remember not to include any formatting when you're pasting content into this block (or any text block, for that matter) because you'll wind up with some goofy inconsistencies. We're also using Option 3 for the background color, which helps break up the page.

Here's some dummy text to fill in some space: Vivamus sit amet quam diam. Proin ut tempus ex. Vivamus interdum lobortis elit, quis aliquam lectus elementum sit amet. Nam in pharetra eros, ut feugiat massa. Sed leo diam, pellentesque eget malesuada commodo, rutrum quis enim. Pellentesque semper neque mi, et efficitur elit hendrerit dictum. Maecenas placerat fringilla libero nec eleifend. Suspendisse magna sem, congue ut mauris vel, imperdiet molestie augue. Nulla finibus elit vel consequat malesuada.

This time, we're breaking it down into two columns, and changing the background color! Separating your text content into multiple columns can help with readability, which this source states is optimal when kept between 9 and 12 words per line. Make sure to select "2 column" from the Appearance tab in the text editor for this look. 

I also added a link in that other block so you can see what they look like - woohoo! You'll notice it opens up in a new window, this is ideal for links that lead the user outside of your site, and can be selected under the "Target" option when creating a link - no need to open up a new window if your link is just leading them to another part of your site, that would be tab madness!

And then there we three! In this example, we've selected "3 column" from the Appearance tab, and changed the background color again. You may also choose to set these rows to the same color as the previous block so they blend together. 

Again, switching up the background color of you content block can be a great way to break up your page into clearer bits and pieces, rather than potentially overwhelming the user with a wall of text run together. 

Here we're using Option 2. These colors are all built into the stylesheet of your site, so they're only customizable if it was requested within your proposal.
